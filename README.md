# Dog Daycare Gantt Chart

A simple HTML/CSS/JavaScript application that displays a Gantt chart for dog daycare scheduling. 

## Features

- Mobile-first design
- Sticky headers and columns for easy navigation
- Morning and evening time slots for each day
- Mock data for demonstration

## Project Structure

- `index.html` - Main HTML file
- `styles.css` - CSS styles for the Gantt chart
- `script.js` - JavaScript file with mock data and rendering logic

## How to Use

1. Simply open the `index.html` file in your web browser
2. Scroll vertically to view different dogs
3. Scroll horizontally to view different days
4. The blue cells represent morning bookings, and the purple cells represent evening bookings

## Mock Data

The app currently uses mock data for demonstration purposes:
- 10 dogs with names
- Sample bookings for January 2023
- Morning (AM) and evening (PM) time slots
